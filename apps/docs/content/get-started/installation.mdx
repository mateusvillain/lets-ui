# Installation

```bash
npm install @lets-ui/styles @lets-ui/components
```

Import the stylesheet and register the components:

```html
<link rel="stylesheet" href="node_modules/@lets-ui/styles/dist/letsui.min.css" />
<script type="module" src="node_modules/@lets-ui/components/dist/index.js"></script>
```
