# Float

Absolutely positions a child element relative to its anchor. Use it to overlay badges, status dots, or any decorative element on top of another component — avatars, cards, icons.

The host element becomes an `inline-block` relative container, so no wrapper is needed.

## With Offset

`offset-x` and `offset-y` shift the floating element from its anchor point. Accepts px values with or without the `px` unit (`"8"` or `"8px"`).
