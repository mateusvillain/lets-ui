### Web Component

```html

  <div>Conteúdo centralizado</div>
</lui-center>
```

### Classe CSS

```html
<div class="center center--horizontal">
  <div class="center__inner" style="max-width: 400px;">Conteúdo centralizado</div>
</div>
```
