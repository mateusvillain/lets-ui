# Image

The `<lui-image>` component wraps a native `<img>` (or `<picture>`) with lazy loading, a skeleton shimmer while loading, an error fallback, and optional aspect-ratio / object-fit controls — preventing layout shift and improving perceived performance.

## Uso

### Web Component

```html
<lui-image
  src="..."
  alt="Descrição"
  width="400px"
  aspect-ratio="16 / 9"
  radius="md"
  fit="cover"
></lui-image>
```

### Classe CSS

```html
<div class="img" style="width: 400px; aspect-ratio: 16 / 9; border-radius: 8px;">
  <img class="img__element" src="..." alt="Descrição">
</div>
```
