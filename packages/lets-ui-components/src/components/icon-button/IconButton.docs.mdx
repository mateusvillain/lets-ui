# Icon Button

Botão compacto que exibe apenas um ícone, sem rótulo de texto. Use quando o espaço é limitado e a ação é reconhecível pelo ícone — como fechar, excluir ou navegar. Sempre forneça um `aria-label` descritivo para garantir acessibilidade.

<Controls of={IconButtonStories.IconButton} />

## Tamanhos

O Icon Button está disponível em dois tamanhos: `lg` (40px) para áreas com mais espaço e `md` (32px) para layouts mais compactos como toolbars e tabelas.

<Canvas of={IconButtonStories.SizeLg} />
<Canvas of={IconButtonStories.SizeMd} />

## Disabled

Use o estado desabilitado quando a ação não está disponível no contexto atual. O botão fica visualmente inativo e não responde a interações.

## Uso

### Web Component

```html
<lui-icon-button size="lg" aria-label="Fechar">
  <svg width="24" height="24" ...>...</svg>
</lui-icon-button>
```

### Classe CSS

```html
<button class="icon-button icon-button--lg" aria-label="Fechar">
  <svg width="24" height="24" ...>...</svg>
</button>
```
