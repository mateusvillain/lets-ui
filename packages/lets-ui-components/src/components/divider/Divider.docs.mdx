# Divider

Separador visual para dividir seções de conteúdo. Suporta orientação horizontal (padrão) e vertical, com ou sem label. Pode ser usado standalone em layouts ou dentro de menus para agrupar itens relacionados.

<Canvas of={DividerStories.Divider} />
<Controls of={DividerStories.Divider} />

## Com label

Use quando precisar de um separador textual entre seções — comum em fluxos de autenticação (ex.: "ou") ou para nomear grupos de conteúdo.

<Canvas of={DividerStories.WithLabel} />

## Vertical

Use `orientation="vertical"` (web component) ou a classe `.divider--vertical` (HTML) para separar itens em layouts horizontais como toolbars e grupos de ações inline.

<Canvas of={DividerStories.Vertical} />

## Uso

### Web Component

```html
<!-- Horizontal -->
<lui-divider></lui-divider>

<!-- Horizontal com label -->
<lui-divider label="ou"></lui-divider>

<!-- Vertical -->
<lui-divider orientation="vertical"></lui-divider>
```

### Classe CSS

```html
<hr class="divider">
<!-- Vertical: -->
<div class="divider divider--vertical"></div>
<!-- Com label: -->
<div class="divider divider--labeled">
  <span class="divider__label">ou</span>
</div>
```
