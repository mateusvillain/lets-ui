## Uso

### Web Component

```html
<lui-inline gap="sm" align="center" justify="start" wrap="wrap">
  <span>Design</span>
  <span>Engineering</span>
  <span>Product</span>
</lui-inline>
```

### Classe CSS

```html
<div class="inline inline--gap-sm">
  <span>Design</span>
  <span>Engineering</span>
  <span>Product</span>
</div>
```
