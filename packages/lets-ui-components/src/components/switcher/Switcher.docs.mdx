# Switcher

Intrinsic responsive layout using the Every Layout switcher pattern. Children switch from horizontal to vertical layout when the container drops below the `threshold` width.

<Controls of={SwitcherStories.Default} />

## Uso

### Web Component

```html
<lui-switcher threshold="320px" gap="md">
  <div>Panel A</div>
  <div>Panel B</div>
  <div>Panel C</div>
</lui-switcher>
```

### Classe CSS

```html
<div class="switcher switcher--gap-md">
  <div>Panel A</div>
  <div>Panel B</div>
  <div>Panel C</div>
</div>
```
