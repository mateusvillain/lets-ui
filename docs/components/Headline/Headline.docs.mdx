# Headline

Componente para títulos e hierarquia textual. Encapsula o par semântico elemento/classe de tipografia, garantindo que o elemento HTML correto seja sempre renderizado para acessibilidade e SEO.

<Canvas of={HeadlineStories.Headline} />
<Controls of={HeadlineStories.Headline} />

## Variantes

Cada variante mapeia para uma escala tipográfica e um elemento HTML semântico padrão.

| Variant       | Tag padrão | Escala       |
| ------------- | ---------- | ------------ |
| `display`     | `div`      | `display`    |
| `title`       | `h1`       | `title`      |
| `subtitle`    | `h2`       | `subtitle`   |
| `headline`    | `h3`       | `headline`   |
| `subheadline` | `h4`       | `subheadline`|
| `block-title` | `h5`       | `block-title`|
| `overtitle`   | `h6`       | `overtitle`  |
