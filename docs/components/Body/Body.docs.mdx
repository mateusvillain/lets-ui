# Body

Componente para texto de conteúdo. Cobre textos de parágrafo, legendas e notas com variações de tamanho, cor, estilo e decoração.

## Tamanhos

| Variant | Tag padrão | Uso sugerido                     |
| ------- | ---------- | -------------------------------- |
| `lg`    | `p`        | Introdução e texto de destaque   |
| `md`    | `p`        | Conteúdo geral (padrão)          |
| `sm`    | `p`        | Legendas, notas e texto auxiliar |
